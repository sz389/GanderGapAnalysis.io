Global Gender Gap Visualization

By Sisi Zhang

Overview

This data visualization is going to analysis the global gender gap from 2016 to 2020. It will visualize the change of the gender gap in healthcare, education, economic and politics. The datasets are download from WORLD BANK and cleaned with excel, the missing values are filled according to the global gender gap report from WORLD ECONOMIC FORUM. I use Tableau as the visualization tool and the website is building based on a free template from TEMPLATED. This project is published on GitHub and Zenodo.
